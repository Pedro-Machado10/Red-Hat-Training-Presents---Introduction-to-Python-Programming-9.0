# 🐍 Introdução à Programação em Python - Red Hat

![python-programming-with-red-hat-ad141-rha-ver-9-0.png]()

Este repositório contém todos os exercícios práticos, projetos e anotações teóricas desenvolvidos durante o curso **Introduction to Python Programming** da Red Hat. O objetivo deste repositório é documentar minha evolução e consolidar os conceitos fundamentais da linguagem.

## 🗂️ Estrutura do Repositório

O projeto está organizado em pastas por módulos. Dentro de cada diretório, você encontrará:
* **`Exercício-XX.py`**: Scripts em Python com a resolução dos desafios propostos.
* **`anotaçõesXX.md`**: Meus resumos teóricos e conceitos-chave do módulo.

## 🚀 O que foi aprendido (Módulos)

Aqui está um resumo prático das habilidades desenvolvidas em cada etapa:

* [**Módulo 1**](./My_work/Chapter%2001/) - Basic Python Syntax: Sintaxe básica, tipos de dados numéricos e de string, e operações iniciais de entrada/saída (I/O).
* [**Módulo 2**](./My_work/Chapter%2002/) - Language Components: Estruturas lógicas de controle de fluxo, como condicionais (if/elif) e laços de repetição (for e while).
* [**Módulo 3**](./My_work/Chapter%2003/) - Collections: Criação, ordenação e manipulação de conjuntos de dados usando listas, tuplas, sets e dicionários.
* [**Módulo 4**](./My_work/Chapter%2004/) - Functions: Encapsulamento de código reutilizável através de funções customizadas, escopo de variáveis, expressões lambda e recursão.
* [**Módulo 5**](./My_work/Chapter%2005/) - Modules: Importação de scripts para organização do projeto e uso dos módulos essenciais da biblioteca padrão (como sys, math e datetime).
* [**Módulo 6**](./My_work/Chapter%2006/) - Classes in Python: Fundamentos da Programação Orientada a Objetos (POO), englobando criação de classes, métodos mágicos, herança e polimorfismo.
* [**Módulo 7**](./My_work/Chapter%2007/) - Exceptions: Prevenção e tratamento de erros no código em tempo de execução (try/except), incluindo a criação de exceções personalizadas.
* [**Módulo 8**](./My_work/Chapter%2008/) - Input and Output: Leitura e escrita avançada de streams de dados e arquivos (texto e binários), além de navegação e manipulação de diretórios do sistema.
* [**Módulo 9**](./My_work/Chapter%2009/) - Data Structures: Estruturas de processamento otimizado utilizando sintaxes concisas como List/Dict Comprehensions, Generators e ordenação paralela.
* [**Módulo 10**](./My_work/Chapter%2010/) - Regular Expressions: Uso do módulo re para buscar, validar, manipular e extrair pedaços de strings com base em padrões complexos de texto.
* [**Módulo 11**](./My_work/Chapter%2011/) - JSON: Integração de sistemas e APIs através da serialização e desserialização de dados formatados em JSON.
* [**Módulo 12**](./My_work/Chapter%2012/) - Debugging: Rastreamento e correção de bugs passo a passo pelo terminal utilizando a ferramenta de depuração oficial do Python (pdb).
* [**Módulo 13**](./My_work/Chapter%2013/) - Assessments: Desafios de código (como o jogo High-Low e a canção 99 Bottles) criados para testar e validar na prática os conceitos de todos os capítulos.
